Maths

Hello! And welcome to the Maths section!

Here, you'll learn all the basics about numbers, and doing maths in Python!

If you already learned other programming languages, numbers intend to not use quotes (""), because if you do use quotes, you'll get a string instead of a number!

In programming languages, numbers usually called "Integer". Altough, the normal integer and the decimal-pointed integer is not called an integer, they're called "Double".

Well, thats enough talking. Lets get to action! In the Introduction page, I typed some code to demonstrate the maths in Python. Well, ain't that too easy? Well, we'll learn more about the operators! In usual maths in school, you learned these symbols: +, -, x, and :. Well, in the computer world, "x" is usually typed into "*", and the ":" into "/". So, it becomes like this: +, -, *, /. I'll give you a code to demonstrate!
cookie = 20;
person = 4;
cookies_per_person = cookie / person;
print(cookies_per_person);
As you can see, I used the "/" operator. Pretty simple, right?

Whats that? You want to know more about the other operators? Great! My hands is tired writing all this non-sense.
Okay, theres more other operators than you might think. Theres ++, +=, and much more. (You can just replace the "+" with the other type of basic operators.)

Now, I'll mention the "++" operator.

The "++" operator is basically the "[number] + 1". You can use it to count scores more easily.

And the "+=" operator, almost the same as "++" operator, only you can use numbers other than one and it uses variables on the first one ([var] += 19).
